desenvolvimento web

o que é desenvolvimento web?

o desenvolvimento web inclui criação,codificação e programção de sites e seus respectivos elementos e linguagens.ele tambem envolve integração com sistemas e ferramentas,armanezamento de bancos de dados, cibersegurança,entre outros,com o objetivo de estruturar cada detalhe de uma pagina da internet.

o que faz um desenvolvedor web?

o desenvolvedor de sites,chamadode profissionalde dev ou web developer, é o responsavel por todo o planejamento estrutural da pagina ultilizando linguagens de programçao como html,css, javascrivit,php, entre varias outras.
